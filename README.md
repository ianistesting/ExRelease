ExRelease
=========

ExRelease is a helper script for screenshot comparisons of softsubbed fansub releases with AvsPmod.


Requirements
------------
ExRelease requires PowerShell 3 and the following software to be available from your PATH environment variable:

* [FFMS2](https://code.google.com/p/ffmpegsource/)
* [MKVToolNix](http://www.bunkus.org/videotools/mkvtoolnix/)
* [FontUtils](http://scripts.sil.org/cms/scripts/page.php?site_id=nrsi&id=FontUtils#fb138857)


Installation
------------
Unpack the ExRelease archive to: _%userprofile%\Documents\WindowsPowerShell\Modules_.


Usage
-----
For more information on ExRelease, command line parameters and examples run the following PowerShell command:

    Get-Help ExRelease
